FOTOBUCH

Redaktionelle Fotobücher aus deiner Mac-Mediathek. AI-Kuration, echte Layouts, druckfertiges PDF für Saal Digital.

Version 0.1.0 laden
macOS 15 · Apple Silicon empfohlen · 3,3 MB

Erste Installation

  1. Zip herunterladen und entpacken.
  2. FOTOBUCH.app in den Ordner Programme ziehen.
  3. Beim ersten Start: Rechtsklick → Öffnen (Gatekeeper-Warnung einmalig bestätigen).
  4. Fotos-Berechtigung freigeben, wenn danach gefragt wird.

Updates

Sobald eine neue Version verfügbar ist, meldet FOTOBUCH das beim nächsten Start. Manuell prüfen über FOTOBUCH → Nach Updates suchen…
